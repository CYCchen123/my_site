📦 0-1 背包问题详解

一、问题定义

给定：

  • 背包最大容量为 W
  • 有 n 个物品，每个物品有重量 w[i] 和价值 v[i]

目标：从这 n 个物品中选择若干个放入背包，使得总重量不超过 W，总价值最大。

注意：每个物品最多只能选一次，不能切割，这就是“0-1”的含义。

二、算法思路：动态规划

1. 状态定义

定义二维数组 dp[i][j] 表示：考虑前 i 个物品，当前背包容量为 j 时的最大价值。

2. 状态转移方程

if j >= w[i]:
    dp[i][j] = max(dp[i-1][j], dp[i-1][j - w[i]] + v[i])
else:
    dp[i][j] = dp[i-1][j]

3. 边界初始化

  • dp[0][j] = 0：不选任何物品，价值为0
  • dp[i][0] = 0：背包容量为0，价值为0

三、算法实现（二维数组）

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int knapsack(int W, const vector<int>& w, const vector<int>& v) {
    int n = w.size();
    vector<vector<int>> dp(n + 1, vector<int>(W + 1, 0));

    for (int i = 1; i <= n; ++i) {
        for (int j = 0; j <= W; ++j) {
            if (j >= w[i - 1])
                dp[i][j] = max(dp[i - 1][j], dp[i - 1][j - w[i - 1]] + v[i - 1]);
            else
                dp[i][j] = dp[i - 1][j];
        }
    }
    return dp[n][W];
}

四、空间优化（滚动数组）

int knapsack(int W, const vector<int>& w, const vector<int>& v) {
    int n = w.size();
    vector<int> dp(W + 1, 0);

    for (int i = 0; i < n; ++i) {
        for (int j = W; j >= w[i]; --j) {
            dp[j] = max(dp[j], dp[j - w[i]] + v[i]);
        }
    }
    return dp[W];
}

注意：容量遍历顺序必须从大到小，否则会使用当前轮已更新的数据，影响正确性。

五、示例说明

物品 重量 w 价值 v
1115
2320
3430

背包容量 W = 4。

最优解：选第 1 和第 2 个物品，总重量 = 1+3=4，总价值 = 35。

六、时间与空间复杂度

  • 时间复杂度：二维 DP 和一维优化均为 O(n × W)
  • 空间复杂度：二维 DP 为 O(n × W)，一维优化为 O(W)

七、应用扩展

  • 完全背包：物品可重复选（内层循环从小到大遍历）
  • 多重背包：每个物品数量有限制
  • 分组背包：物品被分为若干组，每组最多选一个

八、总结
